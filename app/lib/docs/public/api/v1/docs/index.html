    Introduction

    API documentation for RAPiDSkills. This API follows the JSON:API v1.0 specification. Clients must send all requests with the header Content-Type: application/vnd.api+json without any media type parameters.

    Authentication

    After successful sign in, an access token is returned in the response. All authenticated calls are required to pass this access token as a Bearer token in the header of the request. The tokens do not expire, but signing out or changing the user password will invalidate the token.

    Sign in

    Example request

    curl -X POST \
  http://www.rapidskills.org/api/v1/sessions \
  -H 'Accept: application/vnd.api+json' \
  -H 'Content-Type: application/vnd.api+json' \
  -d '{
    "data": {
        "type": "sessions",
        "attributes": {
            "email": "mickey@example.com",
            "password": "supersecret"
        }
    }
}'

    Pagination

    Example request

    curl -X GET http://www.rapidskills.org/api/v1/occupation_standards?page[number]=3&page[size]=10 \
  -H 'Accept: application/vnd.api+json' \
  -H 'Content-Type: application/vnd.api+json' \

    Remove category from work process

    User must own the work process from which the category is being removed, and the category must belong to the work process. Categories with nested skills cannot be removed.

    Example request

    curl --location --request DELETE 'http://www.rapidskills.org/api/v1/work_processes/6/relationships/categories' \
--header 'Accept: application/vnd.api+json' \
--header 'Content-Type: application/vnd.api+json' \
--header 'Authorization: Bearer eyJhbGciOiJIUzI1NiJ9.eyJpZCI6MjIsImVuY3J5cHRlZF9wYXNzd29yZCI6IiQyYSQxMSRXQlFKZmZ6WGFzVTBkUHZBTG1SQWZ1LkZ0TExKWE90MjBsdXQyakdxa2lKVHQ1WVYyOEt0SyIsInNlc3Npb25faWRlbnRpZmllciI6IjBhZDgyNDA5LTJiYWQtNDBhOS1hMjg2LWViMWViMzA3YTIwYyJ9.UvgXioiZAvm6Pts_1DolG6MN_u1ZG3HnQZopGJFpkE0' \
--data-raw '{
    "data": [
        {
            "type": "category",
            "id": "1"
        },
        {
            "type": "category",
            "id": "2"
        }
    ]
}'

    HTTP Request

    DELETE http://www.rapidskills.org/api/v1/work_processes/:work_process_id/relationships/categories

    Query Parameters

    Array specifying type and id must be passed.

    Parameter Type Description
    type
    required    		 string "category"
    id
    required    		 string ID of category to remove from work process

    Returns HTTP status 204, no content.
